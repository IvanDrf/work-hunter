# Branches

## Naming
Branch name must have following structure
```bash
git branch direction_purpose
```

directions:
- front (frontend)
- back (backend)
- ml (machine learning)

purpose:
- working on service/feature
- tests
- docs     
- fix 
- ref 


examples:
```bash
git branch back_auth
git branch front_account
git branch front_docs
git branch back_tests
```
